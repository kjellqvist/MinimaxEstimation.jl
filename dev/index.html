MinimaxEstimation

A Julia package for reproducing and experimenting with the results from the article Minimax Adaptive Estimation for Finite Sets of Linear Systems. https://arxiv.org/abs/2103.02479

Installation

using Pkg;
Pkg.add(url="https://github.com/kjellqvist/MinimaxEstimation.jl")

and activate the environment by running

Pkg.activate("MinimaxEstimation")
